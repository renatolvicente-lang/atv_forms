O que é ?

Os formulários HTML são a principal forma pela qual os sites coletam informações do usuário e as enviam para um servidor para processamento.São essenciais para funcionalidades como logins, cadastros, barras de pesquisa e páginas de contato.







Utilidades dos Formulários no HTML

Normalmente ultilizado para paginas de de cadastro, paginas de login, paginas de recuperação, formularios em si ou até calculadoras e é uma das pricipais ferramentas para coletar dados de usuarios e enviá-los para o banco de dados.

Elementos dos Formulários

Para fazermos um formulario em HTML ultilizamos a tag < form > que é onde iremos colocar tags com input(normalmente usado para botar botões e coisas que o usuario pode interagir), label(usado como um rotulo), textarea(normalmente usado para caixas de texto), select(define uma lista suspensa) e option(elementos da lista suspensa) e buttonpara botões).

Atributos principais

Os formularios podem receber atributos como o action = "" que é ultilizado para definir uma ação para o formulario, o atibuto method = "" é utilizado para expecificar o metodo HTML ultilizado para enviar dados do formulario e possui dois valores validos o GET(Os dados são anexados à URL como pares nome/valor) e o POST(Os dados são enviados no corpo da requisição HTTP).







Organização dos Formularios

Uso correto do label

Como o o lable funciona como um rotulo o jeito certo seria botar ele acima de campos onde o usuario tem que executar algo para que o usuario saiba o que fazer com aquilo.

Agrupamentos de campos

Quando temos um formulário longo ou complexo em seções distintas utilizamos a tag < fieldset > que organiza visualmente e logicamente elementos relacionados, melhorando a experiencia de usabilidade e acessibilidade do formulário

Semântica Básica

A semântica básica em formulários HTML foca em agrupar logicamente os elementos e associar corretamente rótulos aos campos, para isso quando quiser fazer uma tabela do melhor jeito possivel recomenda-se o uso de tags semânticas ditas anteriormente como < form >, < fieldset >, < label > e o < input >, ao invez de divs.
