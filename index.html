Conceitos de Formularios em HTML

O que é? Para que serve? Elementos Semântica




O que é ?

Os formulários HTML são a principal forma pela qual os sites coletam informações do usuário e as enviam para um servidor para processamento.São essenciais para funcionalidades como logins, cadastros, barras de pesquisa e páginas de contato.







Utilidades dos Formulários no HTML

Normalmente ultilizado para paginas de de cadastro, paginas de login, paginas de recuperação, formularios em si ou até calculadoras e é uma das pricipais ferramentas para coletar dados de usuarios e enviá-los para o banco de dados.







Elementos dos Formulários

Para fazermos um formulario em HTML ultilizamos a tag < form > que é onde iremos colocar tags com imput ou ate mesmo label.

Atributos principais

Os formularios podem receber atributos como o action = "" que é ultilizado para definir uma ação para o formulario, o atibuto method = "" é utilizado para expecificar o metodo HTML ultilizado para enviar dados do formulario e possui dois valores validos o GET(Os dados são anexados à URL como pares nome/valor) e o POST(Os dados são enviados no corpo da requisição HTTP).
